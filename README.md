#  Book Club - Login & Register Page

## 📌 Project Description

This project is a simple and modern Login & Register page for a Book Club website developed using HTML5, CSS3, and JavaScript. The project includes a responsive layout and form validation.

# 🌿 Git Workflow

1. The Team Leader created the GitHub repository and add members and create issues(task) for members and tag them.
2. Team members cloned the repository.
3. Each member created a separate branch.
4. Each member completed the assigned task.
5. Changes were added using `git add`.
6. Changes were committed using `git commit`.
7. Branches were pushed to GitHub using `git push`.
8. Each member created a Pull Request.
9. The Team Leader reviewed and merged all Pull Requests into the `main` branch.
